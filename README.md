 Project Description
This project is a 5-page professional portfolio website built using semantic HTML5.

 Pages Included
- About (about.html)
- Contact (contact.html)
- Data (data.html)
- Gallery (gallery.html)
- Home (index.html)
 
 Features
- Semantic HTML structure
- Accessible multimedia (images + video)
- Complex data table
- HTML5 form validation
- Organized file structure

 Folder Structure
- /images → project images
- /videos → video content
- /design → sitemap and wireframes

 How to Run
Open `index.html` in a web browser.
